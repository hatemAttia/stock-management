# 📦 StockPro – Stock & Sales Management

A full-stack **Node.js + HTML** application to manage your inventory, purchases, sales, and generate daily reports — with a modern dark UI and interactive charts.

---

## 🚀 Getting Started

```bash
npm install
npm start
```

Open → **http://localhost:3000**

---

## ✨ Features

| Module | Description |
|---|---|
| 🏠 **Dashboard** | KPI cards (revenue, cost, profit, today's stats), charts, low-stock alerts |
| 🏷️ **Products** | Add / delete products with custom units |
| 🛒 **Purchases** | Declare stock purchases with cost tracking, filter by date |
| 💰 **Sales** | Declare sales with automatic stock validation, filter by date |
| 📊 **Stock** | Real-time remaining stock per product with status badges |
| 📈 **Reports** | Daily revenue vs cost table + grouped bar chart |

---

## 🗂 Project Structure

```
stockmanagement/
├── server.js          ← Express API + lowdb persistence
├── db.json            ← Auto-generated JSON database
├── package.json
└── public/
    ├── index.html     ← Single-page app shell
    ├── style.css      ← Dark modern theme
    └── app.js         ← Frontend logic + Chart.js charts
```

---

## 🛠 Tech Stack

- **Backend**: Node.js, Express, lowdb (JSON file DB — no compilation needed)
- **Frontend**: Vanilla HTML/CSS/JS, Chart.js
- **No database server required** — data is stored in `db.json`
